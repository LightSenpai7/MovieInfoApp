```bash

```

## üöÄ Getting Started

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd movie-info-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
- Start the development server:
  ```bash
  npm start
  ```
- Run on Android:
  ```bash
  npm run android
  ```
- Run on iOS:
  ```bash
  npm run ios
  ```
- Run on Web:
  ```bash
  npm run web
  ```
